# OpenDCS


## Installation

Build it manually:

    $ git clone https://github.com/geoffjay/opendcs.git
    $ cd opendcs
    $ gem install bundler # if it's not already installed
    $ bundle
    $ rake build
    $ gem install pkg/opendcs-<version>.gem

Or install it from rubygems with:

    $ gem install opendcs
